# Style-Profile

Diese Profile gelten fuer dieses Buchpaket.
Der Dateiname ohne `.md` ist der technische Stil-Slug und wird auch als
Output-Ordner verwendet.

Beispiel:

```text
books/anna-karenina/styles/stil-01-original.md
books/anna-karenina/work/scenes/de/stil-01-original/001/scene-01.md
```

Die Dateien duerfen fuer Experimente frei gekuerzt, kopiert und angepasst
werden. `logic/` bleibt Original-Regelmaterial.
