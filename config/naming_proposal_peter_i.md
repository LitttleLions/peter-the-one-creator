# Namensschreibweise – Vorschlag für „Peter der Erste"

Das Buch enthält sehr viele historische Personen. Hier ein Vorschlag
in drei Stilen, damit du eine Wahl hast.

## Stil A – Russische Transliteration (Duden-Standard)

- Пётр → **Peter**
- Иван → **Iwan**
- Софья → **Sofja**
- Алексей → **Alexei**
- Толстой → **Tolstoi**
- Фёдор → **Fjodor**
- Василий → **Wassili**
- Артамон → **Artamon**
- Борис → **Boris**
- Михаил → **Michail**
- Матвеев → **Matwejew**
- Меньшиков → **Menschikow**
- Голицын → **Golizyn**
- Софья Алексеевна → **Sofja Alexejewna**
- Пётр Алексеевич → **Peter Alexejewitsch**

Vorteile: konsequent, in der deutschen Literatur etabliert, lässt die
russische Herkunft erkennen.

Nachteile: manche Leser erkennen Namen wie „Iwan" oder „Wassili" als
veraltet; bei Historikern teils anders geschrieben.

## Stil B – Deutsche Namensformen (eingedeutscht)

- Пётр → **Peter** (gleich)
- Иван → **Johannes / Johann**
- Софья → **Sophie**
- Алексей → **Alexius**
- Толстой → **Tolstoi** (gleich, Eigenname)
- Фёдор → **Theodor / Fjodor**
- Василий → **Basilius / Wassil**
- Артамон → **Artemon**

Vorteile: leichter zugänglich für deutsches Publikum.

Nachteile: bricht die historische Atmosphäre, ist uneinheitlich,
weil nicht alle russischen Namen deutsche Äquivalente haben.

## Stil C – Russische Originalschrift + Übersetzung in Klammern

- **Pjotr I.** (Peter der Erste) – aber nur bei der ersten Erwähnung,
  danach **Peter**
- **Iwan Artjomjewitsch Browkin** (später) → **Iwan Browkin**
- **Sofja Alexejewna** → **Sofja**

Vorteile: historisch korrekt, transparent.

Nachteile: für den Lesefluss sperrig.

## Empfehlung: **Stil A – Russische Transliteration (Duden-Standard)**

Das ist in deutschen Übersetzungen russischer Literatur der häufigste
Stil (Korolenko, Gogol, Tschechow, Tolstoi erscheinen so in den
Standardausgaben). Es wahrt die Atmosphäre und ist lesbar.

Ausnahmen:
- **Peter der Große** (statt „Peter I.") – als eingebürgerter deutscher Name
- **Sankt Petersburg** (Stadt) – offizieller deutscher Name
- **Moskau, Nowgorod, Archangelsk** usw. – deutsche geographische Namen
