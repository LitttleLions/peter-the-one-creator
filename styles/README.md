# Style-Profile

Dieser globale Ordner enthaelt Vorlagen fuer neue Buchpakete.
Produktive Profile liegen in `books/<book-id>/styles/`.

Der Dateiname ohne `.md` ist der technische Stil-Slug und wird auch als
Output-Ordner verwendet.

Beispiel:

```text
books/peter-i-buch-01/styles/stil-03-branderson.md
books/peter-i-buch-01/work/scenes/de/stil-03-branderson/003/scene-05.md
```

Die Dateien duerfen pro Buch und Experiment frei gekuerzt, kopiert und
angepasst werden. `logic/` bleibt Original-Regelmaterial und soll nicht als
Arbeitsflaeche fuer Stilvarianten benutzt werden.
